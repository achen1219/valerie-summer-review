# Valerie's Summer Review

One site for summer math and Chinese review. Hosted via GitHub Pages.

- `math/` — 6-week math curriculum, 30 days, auto-graded short-answer tests.
- `chinese/` — 6-week Chinese (生字語詞 + 成語) review: slide decks, videos, auto-graded multiple-choice quizzes.
- `scripts/parse_quizzes.py` — regenerates `chinese/data.js` from the NotebookLM quiz markdown exports.

Progress is saved locally in the browser (`localStorage`) — nothing is sent to a server.
